Architecture

Claude Code orchestrates multiple MCP servers to handle browser automation and email integration.

Claude Code (Opus 4.5) │ ├──▶ Playwright MCP │ └── Browser automation │ └── Form filling │ └── File uploads │ └── Screenshot capture │ ├──▶ Gmail MCP │ └── OAuth2 authentication │ └── Search emails │ └── Read verification codes │ └──▶ Local Filesystem └── Resumes (PDF/MD) └── Cover letters └── Configuration
🌐

Browser Automation

Playwright MCP enables Claude to navigate websites, click buttons, fill forms, and upload documents - all through natural language commands.

📧

Email Integration

Gmail MCP with OAuth2 allows Claude to search emails and retrieve verification codes automatically - zero manual copy-paste.

📄

Document Access

Direct filesystem access to resumes and cover letters. Claude reads your documents and intelligently maps content to form fields.

🚀

Agentic Workflow

Multi-step reasoning with tool orchestration. Claude plans the application strategy and adapts to different job site formats.
